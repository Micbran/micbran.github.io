Programming Calculator in JavaFX

A project assigned in my Computer Science II course, this programming calculator not only has some of the Windows 10 look and feel but also all of the functions of the Windows 10 programming calculator.

Details

In the two month period this project was assigned, I built a fully functional programming calculator using a combination of Java (with JavaFX) and CSS (for the styling). For the project, we were given a choice between using Swing graphics or JavaFX graphics (though our professor gave us no in-lecture guidance on either of these) and seeing as how I already knew Swing graphics but didn't know JavaFX, I chose JavaFX. This project was a solo project and most of the challenge of the project was the initial set-up, layout and framework for the actual calculator, as most of that required an intricate knowledge of JavaFX.
